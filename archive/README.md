# archive/

Volatile working notes — hand-offs, investigation trails, one-off incident write-ups — that have
been **acted on and decanted** into the durable docs (`patches/`, `CHANGELOG.md`,
`docs/patch-targets.md`, `scripts/analyse-woocommerce.md`).

These files are kept only for provenance and for any site-specific operational follow-ups that don't
belong in the reference docs. Nothing here is load-bearing: the repo is fully understandable without
it. **Prune freely** once a note's follow-ups are closed.

Convention: prefix an integrated note with `RESOLVED-` and the date it was folded in.
